# 🩺 GlucoSense AI

> Early Diabetes Risk Prediction using Comparative Machine Learning Algorithms

##  About
GlucoSense AI is a real-time web application built as part of my thesis:
**"Comparative Analysis of Machine Learning Algorithms for Early Disease Prediction"**

Users can input their health metrics and instantly receive a diabetes risk prediction
powered by a trained Random Forest model.

##  Algorithms Compared
| Algorithm | Test Accuracy |
|---|---|
| ✅ Random Forest | 77.92% (Best) |
| Decision Tree | 75.97% |
| KNN | 75.32% |
| SVM | 74.03% |
| Logistic Regression | 70.78% |

## 📊 Dataset
- **Source:** Pima Indians Diabetes Database (Kaggle/UCI)
- **Records:** 768 patients
- **Features:** 8 medical indicators

## 🛠️ Tech Stack
- **ML:** Python, scikit-learn, pandas, numpy
- **Backend:** Flask API
- **Frontend:** HTML, CSS, JavaScript
